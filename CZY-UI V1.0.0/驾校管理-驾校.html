数据来源：城市站管理员，通过添加\编辑驾校所得到的的数据

驾校管理列表

表单展示的字段：

序号(从1开始，有小到大排序)、

驾校图片、

驾校名称、

驾校性质（总校、分销）、

负责人、

联系电话（15011002200）、

入驻时间（2020年12月12日 13时23分23秒）、

推荐（推荐排序：设置排序后 前台展示规则：根据排序由小到大排序，排序相同时，根据入驻时间倒序排序）、

操作（

驾校管理 相当于之前的店铺管理（点击跳转到驾校管理后台）、

班型管理（点击跳转到该驾校的班型管理页面）、

教练管理（点击跳转到该驾校的教练管理页面）、

编辑（点击跳转到该驾校的编辑页面）、

删除

）


排序规则：优先展示推荐驾校，根据排序由小到大排序、最后根据浏览量排序由大到小



推荐状态开启，方才展示排序输入框；



*设置推荐排序，展示在用户端驾校列表的推荐排序，未设置推荐排序的，根据驾校的浏览量进行排序

驾校入驻管理

数据来源：用户在前台通过提交驾校入驻表单所得到的数据

列表展示的字段（列表名称：驾校入驻记录）：

序号(从1开始，有小到大排序)、

信息来源（活动名称）、

驾校全称（驾校全称）、

驾校简称、

驾校性质（总校、分校）、

位置（省市县街道+详细地址）、

负责人姓名、

手机号、

状态（未联系、有意向、无意向、入驻成功）、

备注、

操作（进入后台、有意向、无意向）


排序规则，优先根据展示未联系、有意向。其次根据申请时间倒序排序
